# 📚 Book Manager CLI

Aplikacja do zarządzania kolekcją książek w terminalu (Python).

## ✅ Funkcje

- Dodawanie książki
- Lista książek
- Wyszukiwanie po tytule/autorze
- Edycja i usuwanie
- Zapis do JSON
- Eksport do CSV
- Testy jednostkowe

## ▶️ Uruchomienie

```bash
python3 main.py
```

## ✅ Testy

```bash
python3 -m unittest test_library.py
```
